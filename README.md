# 🎯 OpenCV ile Lazerli Nişan Alma Sistemi

Bu proje, OpenCV kütüphanesi kullanılarak gerçek zamanlı bir lazer noktasını takip etmek ve koordinatlarını belirlemek amacıyla geliştirilmiştir.

## 🚀 Proje Hakkında
Özellikle İnsansız Kara Araçları (İKA) ve hedefleme sistemlerinde kullanılmak üzere tasarlanan bu algoritma:
* Kameradan gelen görüntüyü işler.
* Belirli bir parlaklık ve renk aralığındaki lazer noktasını tespit eder.
* Noktanın merkez koordinatlarını anlık olarak hesaplar.

## 🛠 Kullanılan Teknolojiler
* **Dil:** Python
* **Kütüphane:** OpenCV (cv2), NumPy
* **Eğitim:** OpenCV University Bootcamp

## 🎓 Sertifika
Bu proje, **OpenCV University Bootcamp** eğitiminde edinilen görüntü işleme teknikleri ile geliştirilmiştir.
